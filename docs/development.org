#+title: Development

* Prerequisites

- Nix with flakes enabled
- direnv (optional)

* Setup

#+begin_src sh
  nix develop   # enters shell with Rust + minizinc + gecode
  direnv allow  # optional automatic shell entry
#+end_src

* Commands

| Command              | Action                          |
|----------------------+---------------------------------|
| ~just build~         | =nix build=                     |
| ~just check~         | TypeScript check + =nix flake check= + cargo check |
| ~just plan~          | Generate plan from data/        |
| ~just gantt path~    | Render a plan file              |
| ~just run~           | plan + text gantt (piped)       |
| ~just run-html path~ | plan + HTML gantt в†’ file        |
| ~just serve~         | Serve web app at localhost:3004 |
| ~cargo check~        | Rust type-check                 |
| ~cargo test~         | Run Rust unit + integration tests |
