#+title: Kitchen Planner

A CLI tool for modelling kitchen resources, recipes, and cooks, then
generating optimized cooking schedules via constraint programming.

* Quick Start

#+begin_src sh
  nix develop
  just plan        # solve + render text Gantt
  just run-html /tmp/plan.html   # render as HTML
#+end_src

* Status

Full pipeline working: RON-based data model → MiniZinc solver (Gecode) →
text/HTML Gantt renderer + interactive web config UI with client-side
validation. Both CLI (schedule | gantt) and web server (just serve) paths
are functional.
